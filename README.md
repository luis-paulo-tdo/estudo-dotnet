# Curso Pensamento Computacional: Fundamentos da Computação e Lógica de Programação

Primeiro curso dentro da carreira.

## 1. Mundo da Computação

### 1.1. Apresentação
- É necessário entender como o mundo da computação funciona para ser capaz de resolver o seus problemas.
- O Pensamento Computacional é uma abordagem muito usada no mercado para se desenvolver programas.
- Com ela, desenvolvemos bons algoritmos, que são estruturados através da Lógica de Programação.
- A Inteligência Artifical é utilizada como um apoio ao desenvolvedor, sem atrapalhar no aprendizado.

### 1.2. Computação Everywhere
- Hoje em dia, existe uma grande diversidade de computadores, logo existe o uso intensivo da computação.
- Em alguns ambientes, a Computação Pervasiva predomina, havendo muito mais computadores do que humanos.
- A ideia maior da computação é calcular ou processar dados que são recebidos de entrada e gerar saídas.
- Computadores são dispositivos capazes de realizar operações lógico-aritméticas sobre os muitos dados.
- Exemplo 1: Se um ar condicionado está com temperatura maior que 25ºC, então desligue o dispositivo.
- Exemplo 2: Comprei um Item A por R$2,00 e o Item B por R$3,00, logo Item A + Item B é igual a R$5,00.
- Temos uma diversidade de aplicações, um imenso volume de dados e uma crescente demanda por soluções.
- Para resolvermos estes problemas, devemos entender como passar instruções para estes computadores.

### 1.3. Papel da Pessoa Desenvolvedora
- A computação está em todo lugar, e as soluções computacionais são criadas por pessoas para pessoas.
- As soluções computacionais que construídas são comumente compostas por softwares e por hardwares.
- Enquanto os hardwares são os dispositivos computacionais, os softwares são os aplicativos.
- Ao contrário dos hardwares, os softwares são construídos por pessoas, e não por outras máquinas.
- A construção de softwares faz parte da economia criativa, que soluciona problemas dos usuários.
- Dentro desta construção, existe uma equipe que colabora para transformar ideias em soluções.
- Dentro desta colaboração, existem processos, técnicas e fluxos que organizam a construção.
- Os softwares construídos geralmente são sistemas compostos por componentes que colaboram entre si.
- O software não é uma "peça única", ele é composto por back-end, front-end, banco de dados, etc.
- O front-end oferece interações com usuários, já o back-end faz o processamento dos dados recebidos.
- Por trás de cada um destes componentes, existem pessoas com expertises específicas trabalhando.
- Estas pessoas compõem um time bem organizado e trabalham numa esteira de desenvolvimento integrado.
- É através deste fluxo de trabalho, que os componentes são desenvolvidos para formar os softwares.
